# 01. Introducción y Especificaciones

## Descripción

FlashPay es una billetera digital móvil híbrida inspirada en Yape, diseñada para transferencias rápidas, pagos de servicios y gestión de finanzas.

## Stack Tecnológico

- **Frontend:** React Native (Expo SDK 52) + TypeScript.
- **Backend:** Node.js + Express.
- **Base de Datos:** MySQL.
- **ORM/Driver:** mysql2.

## Características Clave

- Autenticación Biocmétrica.
- Transferencias QR y por número.
- Metas de Ahorro.
- Modo Oscuro/Claro (Prepared).
