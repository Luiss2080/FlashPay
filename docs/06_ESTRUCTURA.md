# 06. Estructura de Directorios

## Raíz

- `src/`: Código fuente del Frontend.
- `backend/`: Código fuente del Servidor API.
- `docs/`: Documentación del proyecto.

## Frontend (`src/`)

- `views/`: Pantallas (Inicio, Operaciones, Perfil).
- `components/`: Elementos UI reutilizables (Botones, Tarjetas).
- `navigation/`: Configuración de rutas.
- `services/`: Conexión con API (`api.ts`).

## Backend (`backend/`)

- `src/controllers/`: Lógica de negocio.
- `src/routes/`: Endpoints de la API.
- `db/`: Scripts SQL de migración y seeders.
