# 04. Dependencias Principales

## Frontend

- `react-native-paper`: Componentes UI Material Design.
- `react-navigation`: Navegación entre pantallas.
- `axios`: Peticiones HTTP.
- `expo-camera` / `expo-barcode-scanner`: Funcionalidad QR.
- `react-native-qrcode-svg`: Generación de QRs.
- `expo-haptics`: Vibración y feedback táctil.
- `lottie-react-native`: Animaciones (opcional).

## Backend

- `express`: Servidor web.
- `mysql2`: Cliente de base de datos eficiente.
- `cors`: Manejo de orígenes cruzados.
- `dotenv`: Variables de entorno.
